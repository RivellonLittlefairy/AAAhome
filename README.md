# AAAhome
# 项目功能介绍
一   提供类似于愿望单的邮件提醒服务。
二   通过收集各单机游戏平台商品数据，提供史低价格信息，发售日期，好评度等详细游戏信息。
三   根据用户愿望单与各游戏标签，为用户提供游戏推荐（未完成）
